Object Oriented JavaScript

In Javascript: new keyword create a object from a constructor function.

For examples see the source file in the Inspect Element

Constructors and **THIS keyword

If we want to create a multiple instances of an object we need a constructor

Prototypes

Prototype allows to add new property to an constructor..Prototypes helps not to flood the properties into the function constructor

Prototypal Inheritance

If we want to use the properties of one constructor into an another constructor (or) if we want to inherit the one object type from other

Object.create

Another way of creating objects instead of prototype inheritance and creating constructors

ES6 classes

Classes works same like constructors and Prototypes under the hood but here we can use methods instead of Prototypes, For example: We can define greeting() method inside the class instead of creating like Person.prototype.greeting. These ES6 classes make code look cleaner and easier to understand
NOTE: In classes we can define static method and assign something, We can use static method without instantiating the object, If we want to have a static value which does'nt change then we can use these

ES6 SUB classes

If we want to inherit the properties from one class to another class we can use subclasess
NOTE: To use static we have use that with the original class created not with the instance or object we created if you do it will throw an error saying static method is not a function
