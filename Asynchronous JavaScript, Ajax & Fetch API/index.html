Synchronos, Asynchronos Programming and AJAX fetch concepts

Synchronous Programming: In Synchronous Programming, a function/method or something has to wait for each of the above operation to complete. In more specific 2nd line of the code will be exceuted only the first line of the code executes

Example:
load syncPost(function(){
//waits till posts are fetched
//perform some operation with the posts
})
doSomeOtherThing();

In the above example doSomeOtherThing() function will not be executed until and unless load syncPost() completes its operations

ASynchronous Programming: In ASynchronous Programming 2nd line of the code will get excuted even the operation of the first line is not completed

Example:
load syncPost(function(){
//waits till posts are fetched
//perform some operation with the posts
})
doSomeOtherThing();

In the above example doSomeOtherThing() function will be executed eventhough load syncPost() not completed its operations

Few ways to handle Asynchronos code is Callbacks, Promises(New feature), Async/wait(New feature)
